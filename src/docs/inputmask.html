Imask

vanilla javascript input mask

Imask documentation

Usage

CSS

Copy-paste the stylesheet <link> into your <head> to load the CSS. 

<link rel="stylesheet" href="@@webRoot/assets/css/theme.min.css">

Js

Copy-paste the following <script> and Function near the end of your pages. 

<script src="@@webRoot/node_modules/imask/dist/imask.min.js"></script>
<script src="@@webRoot/assets/js/vendors/inputmask.js"></script>
